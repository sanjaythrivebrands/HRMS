## Quick Reference

### Tech Stack
- Frontend: Next.js 14 + TypeScript + Tailwind + ShadCN/UI
- Backend: Node.js + Express + TypeScript
- Data: Mock/In-memory (ready for DB)

### Main Modules
1. Dashboard
2. Employees
3. Organisation Chart
4. Payroll
5. Attendance & Leave
6. Recruitment
7. Reports
8. Settings
